\subsection{Detailed Description}
All the definitions and parameters. This file has all the configurations, definitions and static parameters which are used in many of the functions of the project. It has also the definition of the \hyperlink{structmyPoints}{my\-Points} structure, which stores all the relevant data about every blob detected (e.\-g. position, size, area, color, orientation).
